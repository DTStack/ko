**ko** 是一个简单而强大的工具，用于管理您的 React 应用程序。

## 特点

* 支持在 **webpack v5** 和 **esbuild** 上构建应用程序
* 自定义 ko，使其完全按照您的应用程序需求工作
* 内置流行的 lint 工具，用于对源代码进行 lint
* 内置支持 TypeScript

## 安装

您可以使用 npm、yarn 或 pnpm 安装 ko：
``` bash
npm install ko --save-dev
# 或者
yarn add ko --dev
# 或者
pnpm add ko --save-dev
```
